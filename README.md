## directions

Runs on arduboy like hardware ex) Arduino Leonardo, Pro Micro

Install the `arduino-cli`

```
brew update
brew install arduino-cli
```

To export compiled binary:

```
sh shell/build.sh
```

To upload compiled binary:

```
sh shell/upload.sh
```
